# Python 代码规范指南

## 1. 基础标准
- 遵循 **PEP 8** 规范。
- 使用 **Black** 进行自动格式化。
- 强制使用 **Type Hints**（类型提示），特别是复杂的图像处理函数。

## 2. 算法与图像处理
- **明确性**: 图像处理的关键参数（如阈值、核大小）必须通过具名变量或配置项定义，严禁硬编码。
- **文档化**: 涉及 OpenCV、PaddleOCR 或 SAM 的复杂算法流程，必须在函数头注详细说明处理逻辑及输入输出张量形状。
- **模块化**: 将 OCR 检测、SAM 分割、OpenCV 预处理解耦为独立的 Service 类。

## 3. 错误处理
- 针对图像读取失败、内存溢出（处理超大图时）或模型加载异常，应使用自定义 Exception 进行捕获并返回给前端清晰的错误信息。
