# Track Specification: Track 1 - 高精标注台底座

## 1. 概述
本赛道旨在构建 InkGrid (墨阵) 的核心渲染与标注引擎。通过 PixiJS 处理超高清碑帖图像，并集成 Paper.js 实现高精度的 Bézier 曲线标注，为后续的 AI 辅助标注提供坚实的交互基础。

## 2. 功能需求
- **超高清图像渲染 (PixiJS)**: 支持加载并平滑缩放/平移大型碑帖图像（支持瓦片化加载思路）。
- **矢量标注核心 (Paper.js)**: 支持绘制、编辑和微调高阶贝塞尔曲线。
- **中英双语 UI 框架**: 搭建符合“赛博金石”美学的双语基础界面。
- **数据结构支持**: 定义能够存储 Bézier 路径、节点和元数据的 JSON 格式。

## 3. 技术栈
- 前端: React 19 + TypeScript + PixiJS + Paper.js
- 样式: Tailwind CSS v4
- 国际化: i18next

## 4. 验收标准
- 能够流畅缩放显示 `峄山碑.jpg`。
- 用户能用贝塞尔曲线完整勾勒出“皇”字的笔画边缘并保存路径。
- 界面支持中英双语切换，且符合“新国风”审美。
