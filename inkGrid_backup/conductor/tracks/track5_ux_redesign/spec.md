# Track Specification: Track 5 - 赛博金石沉浸式 UX 重构

## 1. 概述
本赛道旨在将 InkGrid 从“工具原型”升级为“专业艺术标注站”。核心目标是提升视觉美感（Cyber-Epigraphy 风格）、优化可读性、并引入阶段化工作流（Phased Workflow），确保在任何缩放级别下布局均稳定可靠。

## 2. 功能与交互需求
- **视觉风格**:
    - 底色: #0c0a09 (Deep Stone)
    - 高亮: #f59e0b (Amber 500)
    - 字体: 标题与单字采用 Noto Serif SC (衬线体)，UI 说明采用 Monospace。
- **阶段化工作流 (Phased Workflow)**:
    - 阶段 1: **识碑 (ID)** - 侧重于碑帖识别与信息展示。
    - 阶段 2: **建阵 (Grid)** - 侧重于网格校准，弱化其他面板。
    - 阶段 3: **对位 (Align)** - 双流联动，侧重于全文比对。
- **布局优化**:
    - 使用 CSS Grid 重构主框架，彻底解决 PixiJS 画布在缩放时撑破容器的问题。
    - 侧边栏字号提升：单字预览 > 4rem，正文 14px-16px，行高 1.6。
- **响应式 Canvas**:
    - 实现 ResizeObserver 监听，确保画布在窗口变化时自动重绘并保持中心对齐。

## 3. 验收标准
- 页面具备明显的“设计感”，符合高端文化工具审美。
- 在不同分辨率下，侧边栏内容不发生重叠或溢出。
- 文字可读性有质的提升。
- 实现至少两个明显的阶段切换（如 Grid 模式与 Inspection 模式）。
