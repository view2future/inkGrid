# 查看原文释义功能 - 使用说明

## ✅ 功能已完成

原文释义功能现已集成到「赏一帖」功能中，包括**桌面端**和**移动端**。

---

## 📱 如何使用

### 移动端（手机/平板）

1. 打开「赏一帖」功能
2. 选择任意碑帖（如《兰亭序》）
3. 在 **「赏析」** 页面（默认页面），向下滚动
4. 在碑帖标题和简介下方，找到红色按钮：

   ```
   ┌─────────────────────────────┐
   │  📖 查看释义                │
   └─────────────────────────────┘
   ```

5. 点击按钮，弹出释义窗口
6. 选择视图模式：
   - **原文**：纯原文展示
   - **释义**：现代文解释
   - **对照**：左右对照学习（推荐）

### 桌面端

1. 打开「赏一帖」功能
2. 选择任意碑帖
3. 在 **「賞析」** 标签页
4. 在赏析内容下方找到红色按钮：

   ```
   ┌─────────────────────────────┐
   │  📖 查看释义    📖 已有释义 │
   └─────────────────────────────┘
   ```

5. 点击按钮，弹出释义窗口

---

## 🎯 释义窗口功能

### 三种视图模式

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| **原文** | 只显示古文原文 | 专注阅读原文 |
| **释义** | 显示现代文解释 | 理解文意 |
| **对照** | 左右分栏对照 | 深入学习 |

### 可折叠区块

- 📖 **创作背景** - 作品创作的历史背景
- ✨ **名句赏析** - 经典名句解读
- 📝 **全文释义** - 完整现代文翻译
- ✏️ **临写指导** - 书法临写建议

## ⚠️ 注意事项

### 开发环境

如果在开发环境中看到 404 错误：

```
GET http://localhost:5173/steles/.../index.json 404 (Not Found)
```

**解决方案**：重启 Vite 开发服务器

```bash
# 停止当前服务器（Ctrl+C）
cd frontend
npm run dev
```

### 生产环境

已构建到 Android 项目，无需额外配置。
