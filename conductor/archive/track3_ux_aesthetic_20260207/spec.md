# Track Specification: Track 3 - 体验升级 (UX & Aesthetic)

## 1. 概述
本赛道致力于将 InkGrid (墨阵) 从功能性工具升华为具备顶级审美的文化艺术产品。通过深度打磨中英双语的“赛博金石”视觉系统，并引入基于行笔感知的游戏化临摹功能，为全球用户提供沉浸式的书法学习体验。

## 2. 功能需求
- **顶级国风美学 UI**: 实现基于玄黑、砗磲白、朱砂红的极简现代布局，加入宣纸纹理与水墨动效。
- **游戏化趣味临摹**: 
  - 实现“节奏大师”式的行笔评价算法，根据重合度、行笔速度提供即时反馈。
  - 增加“墨洇”视觉特效，模拟真实笔墨感。
- **中英双语无缝切换**: 优化 i18next 集成，确保专业术语在两种语言下的文化准确性。
- **AR 寻踪原型 (Web 版)**: 模拟移动端 AR 匹配场景，点击“印章”即可唤起历史背景。

## 3. 技术栈
- 前端渲染: PixiJS + GSAP (动效)
- 样式: Tailwind CSS v4 + 玻璃拟态
- 评价算法: 基于路径相似度匹配 (DTW 或类似算法)

## 4. 验收标准
- UI 设计通过“赛博金石”美学自审，达到行业顶尖水准。
- 临摹功能支持对“皇”字的轨迹评分，得分逻辑符合行笔习惯。
- 中英双语切换流畅，英文术语作为美学点缀不冲突。
