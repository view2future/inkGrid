# Track Specification: Track 2 - AI 语义辅助

## 1. 概述
本赛道旨在为 InkGrid (墨阵) 引入智能化标注能力。通过集成 SAM 实现笔画级的自动分割，并利用预设的碑文真值数据库实现图像与文字的自动对齐，从而极大提升标注效率和准确性。

## 2. 功能需求
- **集成 SAM (Segment Anything Model)**: 支持后端推理，接收前端坐标并返回精准的笔画遮罩。
- **碑文真值自动对齐**: 根据用户输入的碑帖名（如“峄山刻石”），自动加载全文，并将识别出的文字块按顺序与原文对齐。
- **语义标注增强**: 在标注界面实时显示当前选中文字的原文、简体、拼音及释义。
- **资产自动化打包**: 自动生成包含 SVG 路径、PNG 裁剪图及 JSON 元数据的标准化资产包。

## 3. 技术栈
- 后端: FastAPI + Python 3.11+
- AI: SAM (vit_b) + PaddleOCR
- 前端集成: React Query + Canvas 同步

## 4. 验收标准
- 用户点击图片中的字，系统能自动生成并显示该字的精准边缘闭合路径。
- 《峄山碑》的前 10 个字能被自动定位并与“皇帝立国维初廿六”一一对应。
- 导出的 JSON 文件包含完整的语义信息。
