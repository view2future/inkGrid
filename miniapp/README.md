# 墨阵 · 微信小程序 (MVP)

目标：在小程序里复刻手机端「墨流」的核心体验，先做可运行的 MVP（Hub + 一字一页）。

## 运行方式

1. 打开微信开发者工具
2. 导入项目：选择本目录 `miniapp/`
3. 在 `miniapp/project.config.json` 里填你的小程序 `appid`
4. 默认使用线上资源：`https://www.inkgrid.art`
   - 如需改成你自己的域名/服务器，编辑 `miniprogram/utils/config.js`

## 注意

- 小程序包体积限制较严格，碑帖大图/单字大量资源建议走 CDN（本 MVP 也默认走线上）。
- 需要在小程序后台配置「request 合法域名」或在开发者工具里关闭 `urlCheck`。

## 已实现
