# InkGrid × 林曦·暄桐教室 合作提案

**项目名称**：墨阵·暄桐数字碑帖研习工具
**合作方**：暄桐教室 × InkGrid项目组
**提案日期**：2025年2月
**提案版本**：V1.0

---

## 合作定位

**核心原则**：InkGrid聚焦碑帖字形分析，以字形、笔顺、笔画为核心，向外延伸至临摹辅助和碑帖资源库。

---

## 一、合作基础

### 1.1 InkGrid核心能力

| 能力 | 说明 |
|-----|-----|
| **字形分割** | 将碑帖按字切分，支持任意精度放大 |
| **笔画提取** | 单字笔画分离，笔顺路径标注 |
| **结构分析** | 重心、比例、收放关系可视化 |
| **对比工具** | 原帖与临作的结构叠加对比 |
| **碑帖资源库** | 经典碑帖的结构化存储与检索 |

### 1.2 林曦教学契合点

林曦老师课程强调"读帖"与"临帖"，核心关注：
- 字形结构分析
- 笔画笔势理解
- 从"形似"到"神似"的渐进

**契合点**：InkGrid提供字形级别的可视化工具，辅助学员"读懂"每一个字。

---

## 二、合作内容

### 2.1 读帖辅助工具

配合林曦老师"先读后临"的教学理念：

```
功能清单

【字形分解】
  • 单字框选：一键提取碑帖中的任意单字
  • 无限放大：看清每个笔画的细节
  • 结构线叠加：重心线、横中线、纵中线

【笔画分析】
  • 笔画分离：每个笔画用不同颜色标注
  • 笔顺路径：用箭头标注书写顺序
  • 提按标记：标注粗细变化的位置

【林曦标注】
  • 可叠加林曦老师的结构讲解文字
  • 标注"主笔"和"收放"要点
```

### 2.2 临摹对比工具

配合"对临"与"背临"练习：

```
功能清单

【模板导出】
  • 米字格/九宫格模板（从原帖生成）
  • 透明覆盖层：临作拍照后与原帖叠加
  • 原大/放大两种模式

【结构对比】
  • 临作与原帖的结构叠加显示
  • 差异热图：用颜色标注偏离程度
  • 评分维度：结构准确度、主笔还原度

【进步记录】
  • 同一字的历史临作对比
  • 进步曲线可视化
```

### 2.3 课程配套碑帖库

配合暄桐课程进度提供数字化碑帖：

| 课程阶段 | 碑帖 | InkGrid资源 |
|---------|-----|------------|
| 篆书入门 | 《峄山刻石》 | 小篆结构库、笔画路径 |
| 隶书学习 | 《曹全碑》《张迁碑》 | 隶书特征对比库 |
| 楷书基础 | 《多宝塔碑》《颜氏家庙碑》 | 颜体结构分析 |
| 行书进阶 | 《圣教序》《苏轼手札》 | 行书笔势库 |
| 小楷专修 | 《灵飞经》《赵孟頫汲黯传》 | 小楷精细分析 |

## 三、技术实现

### 3.1 核心功能

| 功能 | 技术方案 |
|-----|---------|
| 字形分割 | SAM模型 + 投影法辅助 |
| 笔画提取 | 边缘检测 + 连通域分析 |
| 笔顺推断 | 基于书写规则的路径生成 |
| 对比分析 | 图像叠加 + 差异像素计算 |
| 模板导出 | SVG矢量格生成 |

### 3.2 数据结构

```
碑帖资源库结构

单字记录：
  ├── 原图切片
  ├── 笔画分割图
  ├── 笔顺路径（坐标序列）
  ├── 结构线（重心、横中、纵中）
  └── 元数据（书体、风格、难度等级）
```
