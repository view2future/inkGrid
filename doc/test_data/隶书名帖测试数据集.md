## 详细描述

### li_001 | 曹全碑
**书法家**: 不可考
**朝代**: 东汉
**类型**: 碑刻
**收藏地点**: 陕西西安碑林
**创作年份**: 东汉中平二年 (185年)

**简介**:
曹全碑全称《汉郃阳令曹全碑》，记载了曹全的生平事迹。此碑于明代万历年间出土，保存完好，字迹清晰，是汉隶中最具代表性的作品之一。

**风格特点**:
- 秀逸舒展，典雅温润
- 笔画波磔分明，节奏感强
- 结体扁平匀称
- 章法疏朗有致
- 笔势流畅婉转

**技术难点**:
- 碑面偶有裂纹
- 部分字口略有漫漶
- 整体保存较好，分割难度低

**IIIF资源**:
- 故宫博物院高清资源
- 西安碑林数字资源

**推荐优先级**: ⭐⭐⭐⭐⭐ (最高，初学者首选)

**界格情况**: 无界格，字距较宽，适合标准分割

## 测试建议

### 初学者推荐优先级 (入门)

1. **曹全碑** - 秀逸舒展，字口清晰，最适合入门
2. **乙瑛碑** - 端庄匀停，法度严谨
3. **史晨碑** - 平和中正，容易把握
4. **礼器碑** - 瘦劲挺拔，笔法精到

### 进阶推荐

1. **张迁碑** - 古拙雄浑，与曹全碑对比学习
2. **石门颂** - 纵放恣肆，摩崖风格
3. **衡方碑** - 浑厚沉雄，气势磅礴
4. **鲜于璜碑** - 近代出土，保存完好

### 高难度挑战

1. **五凤刻石** - 西汉早期，篆隶过渡
2. **石门颂** - 摩崖风化，分割困难
3. **封龙山颂** - 摩崖刻石，保存状况不佳

## 格内碑帖测试建议

隶书碑帖多为无界格碑刻，但部分清代隶书作品有界格形式。若需测试 Grid-to-VLM 方案，建议：

1. **优先选择楷书格内碑帖**（如《九成宫醴泉铭》实际有界格）
2. 隶书格内碑帖较少，可考虑：
   - 清代馆阁体隶书
   - 当代书法家界格隶书作品
   - 影印本隶书字帖
