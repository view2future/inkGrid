## 详细描述

### zhuan_001 | 泰山刻石
**书法家**: 李斯（传）
**朝代**: 秦
**类型**: 碑刻
**收藏地点**: 山东泰安岱庙
**创作年份**: 秦始皇二十八年 (前219年)

**简介**:
泰山刻石是秦始皇东巡登泰山时所刻的纪功碑，由李斯书写。此碑是小篆的典范之作，被誉为"千古第一小篆"。原碑现存约9字。

**风格特点**:
- 法度严谨，结构规范
- 笔画匀称圆润
- 结体端庄秀丽
- 左右对称，疏密有致
- 用笔藏锋逆入，圆起圆收

**技术难点**:
- 原碑仅存残石
- 传世拓本稀少
- 字迹漫漶严重
- 分割难度极高

**IIIF资源**:
- 故宫博物院藏善本拓片

**推荐优先级**: ⭐⭐⭐⭐⭐ (篆书学习必读)

**篆书类型**: 小篆
**界格情况**: 碑刻自然分布，无界格

## 篆书识别特殊难点

### 1. 字形演变差异大

| 时代 | 字形特点 | 识别难度 |
|-----|---------|---------|
| 甲骨文 | 象形性强，笔画瘦劲 | ★★★★★ |
| 西周金文 | 笔画浑厚，结体端庄 | ★★★★☆ |
| 春秋战国 | 风格多样，地域差异大 | ★★★★★ |
| 秦小篆 | 法度统一，结构规范 | ★★★☆☆ |
| 清代篆书 | 融汇古法，风格多变 | ★★★★☆ |

### 2. 载体差异

| 载体 | 特点 | 处理建议 |
|-----|-----|---------|
| 青铜器 | 立体铭文，需特殊光照 | 多角度拍摄 |
| 甲骨 | 字形微小，刻辞分散 | 超高分辨率 |
| 石刻 | 风化漫漶，拓本差异 | 选择精良拓本 |
| 墨迹 | 笔势清晰，书写性强 | 直接识别 |

### 3. 识别策略建议

```
篆书识别推荐流程:

1. 图像预处理
   ├─ 青铜器：多角度拍摄、光照补偿
   ├─ 甲骨：超高分辨率(1200dpi+)、去噪
   └─ 石刻：选择精良拓本、二值化处理

2. 字形归类
   ├─ 判断时代：甲骨/金文/小篆
   ├─ 判断载体：青铜/甲骨/石刻/墨迹
   └─ 建立对应字典

3. 识别模型
   ├─ 甲骨文：专用甲骨识别模型
   ├─ 金文：融合金文特征库
   ├─ 小篆：基于Unicode标准字符
   └─ 清篆：结合书家风格特征

4. 后处理
   ├─ Unihan数据库映射
   ├─ 异体字关联
   └─ 专家纠错机制
```
