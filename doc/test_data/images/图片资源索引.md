## 图片下载脚本

### 批量下载脚本 (download.sh)

```bash
#!/bin/bash

# 创建目录
mkdir -p images/楷书
mkdir -p images/隶书
mkdir -p images/篆书

# 下载曹全碑 (已验证可用)
echo "下载曹全碑..."
curl -L -o images/隶书/cao-quan-bei-01.jpg \
  "https://commons.wikimedia.org/wiki/Special:FilePath/%E6%9B%B9%E5%85%A8%E7%A2%91-%E4%B8%9C%E6%B1%89%E4%B8%AD%E5%B9%B3%E4%BA%8C%E5%B9%B4%EF%BC%88185%EF%BC%89-%E8%A5%BF%E5%AE%89%E7%A2%91%E6%9E%97%E7%AC%AC%E4%B8%89%E5%AE%A4_2023-09-29_01.jpg"

echo "下载完成!"
```

**使用方法**:
```bash
chmod +x download.sh
./download.sh
```

| 故宫博物院 | https://www.dpm.org.cn/iiif | 部分开放 |
| 台北故宫 | https://iiif.npm.edu.tw | 需申请 |
| 大都会博物馆 | https://www.metmuseum.org/iiif | 开放获取 |
| 斯坦福大学 | https://sudiva.stanford.edu/iiif | 开放获取 |

### IIIF 图片请求示例

```bash
# 获取完整图
https://iiif.example.com/identifier/full/full/0/default.jpg

# 获取指定区域
https://iiif.example.com/identifier/{x},{y},{w},{h}/full/0/default.jpg

# 获取指定尺寸
https://iiif.example.com/identifier/full/500,/0/default.jpg
```
