### 五言绝句（20 字）
```
1. 白日依山尽，黄河入海流。欲穷千里目，更上一层楼。
2. 床前明月光，疑是地上霜。举头望明月，低头思故乡。
3. 春眠不觉晓，处处闻啼鸟。夜来风雨声，花落知多少。
4. 红豆生南国，春来发几枝。愿君多采撷，此物最相思。
5. 空山不见人，但闻人语响。返景入深林，复照青苔上。
```

### 七言绝句（28 字）
```
1. 两个黄鹂鸣翠柳，一行白鹭上青天。窗含西岭千秋雪，门泊东吴万里船。
2. 朝辞白帝彩云间，千里江陵一日还。两岸猿声啼不住，轻舟已过万重山。
3. 日照香炉生紫烟，遥看瀑布挂前川。飞流直下三千尺，疑是银河落九天。
4. 故人西辞黄鹤楼，烟花三月下扬州。孤帆远影碧空尽，唯见长江天际流。
5. 月落乌啼霜满天，江枫渔火对愁眠。姑苏城外寒山寺，夜半钟声到客船。
```

```
1. 国破山河在，城春草木深。感时花溅泪，恨别鸟惊心。
   烽火连三月，家书抵万金。白头搔更短，浑欲不胜簪。

2. 空山新雨后，天气晚来秋。明月松间照，清泉石上流。
```

### 二、书体知识

#### 简答题
```
1. 简述楷书的主要特点。
   参考答案：楷书笔画平直，结构方正，章法严谨，易于辨认。
   横平竖直，点画分明，是学习书法的基础书体。

2. 隶书与篆书的主要区别是什么？
   参考答案：
   - 隶书笔画有波磔，篆书笔画圆润均匀
   - 隶书结构扁平，篆书结构修长
   - 隶书有蚕头燕尾，篆书无此特征
   - 隶书便于书写，篆书装饰性强

3. 行书与草书的区别是什么？
   参考答案：
   - 行书介于楷书与草书之间，草书更为简省
   - 行书易于辨认，草书较难识别
   - 行书笔画连贯适度，草书笔画高度简化
   - 行书实用性强，草书艺术性更强
```

### 四、章法与创作

#### 论述题
```
1. 简述书法作品的章法布局要点。
   参考答案：
   - 正文与落款的比例协调
   - 字距、行距的合理安排
   - 留白的艺术处理
   - 落款位置与内容规范
   - 钤印的位置与数量
   - 整体气韵的统一

2. 书法创作中如何处理"意在笔先"？
   参考答案：
   - 创作前充分理解文字内容
   - 构思整体章法布局
   - 确定书体风格
   - 预想字形结构
   - 调整心态，凝神静气
   - 胸有成竹后再动笔
```

2. 临摹时注意观察原帖的笔画特征和结构特点
3. 创作练习注重章法完整性
4. 学习基础书法理论知识
5. 模拟考试环境，控制创作时间

### 8 级备考要点
1. 精通主修书体，兼习其他书体
2. 提高临摹精准度，追求形神兼备
3. 加强创作能力训练，形成个人风格
4. 系统学习书法史和书法理论
5. 了解文房四宝知识
6. 多观摩名家作品，提高审美能力
