# inkGrid - 部署到 Zeabur

## 重要说明

此版本的 inkGrid 应用包含完整的前端界面，包括：
- 首页（5个篆体字轮转）
- 左下角墨廊入口
- 右下角峄山刻石赏析入口
- 完整的 SPA 功能

## 部署配置

### 在 Zeabur 控制台中：

1. **创建服务**：
   - 选择 "Docker" 部署方式
   - 使用默认的 `Dockerfile`

2. **端口设置**：
   - 端口：8000（Dockerfile 中已配置）
   - 协议：HTTP

3. **健康检查**：
   - 路径：`/health`
   - 端口：8000

4. **环境变量**：
   - Zeabur 会自动设置 `$PORT` 环境变量
   - 应用会自动使用此端口

### 架构说明

此 Docker 镜像包含：
- **前端**：React SPA，构建产物由 Nginx 服务
- **后端**：FastAPI 服务，运行在内部端口 8001
- **Web 服务器**：Nginx，运行在端口 8000，提供前端页面并代理 API 请求到后端

### 访问应用

部署完成后，您将能够：
- 访问首页，看到5个篆体字轮转
- 点击左下角墨廊入口进入画廊
- 点击右下角峄山刻石赏析入口进入专题页面
- 使用所有前端功能

### 故障排除

如果仍然看到后端信息而不是前端界面：
1. 检查健康检查路径是否设置为 `/health`（而不是 `/`）
2. 确认端口设置为 8000
3. 等待部署完成（构建可能需要几分钟时间）
